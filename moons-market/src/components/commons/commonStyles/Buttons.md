# 버튼 이름 규칙

- hover가 없는 경우: <버튼색상 Button>
- hover가 있는 경우: <기본색상 hover색상 Button>

# text(선택 인자)

- 지정된 style이 다른 경우 text로 버튼명을 전달해주세요.

---
